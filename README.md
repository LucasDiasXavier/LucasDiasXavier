## 👨‍💻 Sobre Mim

Há um momento na vida em que você olha para a tela do computador e pensa: **"Eu quero criar isso."**

Esse momento chegou para mim — e mudou tudo.

Sou **Lucas Dias Xavier**, e estou construindo minha história na programação a partir do zero, com cursos online, muita disciplina e uma vontade genuína de evoluir. Não tenho um diploma de TI, não cresci programando — mas tenho algo que nenhuma grade curricular ensina: a **determinação de quem escolhe recomeçar e não olha para trás.**

Cada repositório aqui é uma página dessa história. Comecei com o básico do HTML, entendi como a web funciona por baixo dos panos, e fui construindo projeto após projeto — uma imobiliária, uma cafeteria, uma pet shop, um site de tecnologia. Não são só exercícios: são **provas de que estou aprendendo de verdade.**

Me motiva criar interfaces que as pessoas tocam e sentem. Me motiva resolver um problema com código que antes parecia impossível. Me motiva a ideia de que, com consistência, posso transformar esse aprendizado em uma carreira sólida em tecnologia.

Ainda tenho muito a aprender — e é exatamente isso que me faz acordar todo dia querendo codar.

```text
🧩 Resolve problemas reais com código
🎨 Apaixonado por interfaces bonitas e funcionais
📚 Aprendizado contínuo via cursos online e projetos práticos
🚀 Objetivo: Desenvolvedor Front-End Jr. → Full Stack
💬 Aberto a feedbacks, mentorias e oportunidades
```
